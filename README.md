# makemcp.dev

Landing page for [@meimakes](https://github.com/meimakes) open source MCP servers.

## Servers

| Server | What It Does | Repo |
|--------|-------------|------|
| strava-mcp-server | Activities, routes, and training data from Strava | [GitHub](https://github.com/meimakes/strava-mcp-server) |
| hevy-mcp-server | Workout history and PRs from Hevy | [GitHub](https://github.com/meimakes/hevy-mcp-server) |
| oura-mcp-server | Sleep, readiness, and HRV from Oura Ring | [GitHub](https://github.com/meimakes/oura-mcp-server) |
| amazon-mcp-server | Search, cart, and orders from Amazon | [GitHub](https://github.com/meimakes/amazon-mcp-server) |

All servers support dual transport (stdio + SSE).

Static HTML hosted on GitHub Pages at [makemcp.dev](https://makemcp.dev).
